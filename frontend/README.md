# AI Chatbot (React + Node.js + RAG)

## Features

- Chat UI (popup style)
- Backend API (Express)
- Simple RAG from local data
- Natural language support
- Chat history (localStorage)

## Tech Stack

- React (Vite)
- Node.js + Express
- JavaScript

## How to run

### Backend

cd backend
npm install
node server.js

### Frontend

cd frontend
npm install
npm run dev
